Outlook 邮箱管理台

本地批量邮箱管理工作台

导入格式

每行一个账号，默认支持以下字段顺序：

email----password----client_id----refresh_token----access_token----group----note
使用说明

仅用于管理你自己拥有或已获授权的 Outlook 邮箱账号，默认在本地运行。

本地部署

统一管理 Outlook 账号、令牌、状态检测与邮件预览

适合本地批量导入、状态检查、收件箱预览和导出备份，操作流程接近后台管理台。

接口服务 本地存储 本机运行
账号总数 0 当前本地数据库中的 Outlook 账号数量
可用账号 0 可正常访问收件箱的账号
待处理 0 检测失败、权限不足或缺少令牌
缺少令牌 0 尚未填写 access_token 或 refresh_token

邮箱控制中心

支持搜索、导入、检测、删除、查看邮件和导出备份。

邮箱账号 分组 接入方式 凭据 状态 最近检测 操作

还没有账号

点击“新增账号”或“批量导入”开始录入 Outlook 邮箱。

账号详情

从左侧选择一个账号后，这里会显示凭据情况、备注、时间线和快捷操作。

新增账号

密码仅用于本地归档记录，真正访问收件箱使用访问令牌或刷新令牌。

批量导入

支持纯文本、CSV 标题行和制表符分隔内容。

示例 
sales01@outlook.com----pass-demo----client-id-demo----refresh-token-demo--------销售组----主账号
sales02@outlook.com--------refresh-token-demo----access-token-demo----售后组----备用账号

邮件预览

显示当前账号收件箱最近 20 封邮件。

邮件预览

从左侧选择一封邮件后，在这里显示正文内容。
